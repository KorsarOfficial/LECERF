all peripheral state; eliminate ALL 6 process-globals (g_cpu_for_scb, g_dwt_for_run,
g_nvic_for_run, g_tt, g_jit_for_tt, g_replay_mode). Add uart_t.sink_ctx field.

board A must not surface in board B).

corrupt each other through different paths -- not just CPU state but also TT event
logs and NVIC pending tables. The Python/Docker/CI stack is unsafe until every
hot-path read of a process-global is replaced with a per-board ctx read.

Static profile table:
  static const board_profile_t BOARDS[] = {
      { "stm32f103",  0x00000000, 128*1024, 0x20000000,  20*1024, CORE_M3, 0 },
      { "stm32f407",  0x00000000, 512*1024, 0x20000000, 128*1024, CORE_M4, 0 },

board_create(name): linear scan profile, calloc(board_t), heap-alloc jit (calloc),
heap-alloc uart_buf (cap=64KB), bus_init, bus_add_flat flash+sram per profile,
attach all peripherals, wire b->scb.ctx = &b->cpu, b->uart.sink = uart_capture,
b->uart.sink_ctx = b. board_create MUST NOT touch g_cpu_for_scb / g_dwt_for_run /
g_nvic_for_run / g_tt / g_jit_for_tt / g_replay_mode -- those globals stay at their
default NULL/false values for the legacy tools/main.c CLI. Run cpu_reset, return
(board_t*)b.

uart_capture(void* ctx, int c): board_t* b = ctx; if (b->uart_buf_n < cap)
b->uart_buf[n++] = (u8)c; return c.

board_flash(b, data, sz): bus_load_blob into flash region, set cpu.msp =
bus_r32(0), cpu.r[REG_SP] = msp, cpu.r[REG_PC] = bus_r32(4) & ~1.

board_run(b, max_steps, &exit_cause): build a stack-local run_ctx_t = { .cpu=&b->cpu,
.bus=&b->bus, .nvic=&b->nvic, .st=&b->st, .scb=&b->scb, .dwt=&b->dwt, .jit=b->jit,
.tt=b->tt, .replay=false }; set bus_set_cookie(&b->bus, &ctx) so peripherals
(uart_read replay path) can fish it out via bus_get_cookie; call run_steps_full_gc
with the ctx. Set *exit_cause = LECERF_HALT if cpu.halted, LECERF_TIMEOUT if cycles
== max_steps reached, LECERF_FAULT if cpu.cfsr | hfsr. Return cycles run.

board_uart_drain(b, dst, cap): n = min(b->uart_buf_n, cap); memcpy; reset
uart_buf_n = 0; return n.

board_gpio_get(b, port, pin): switch port { case 0: bit = (b->stm32.odr_a >> pin)
& 1; ... 2: odr_c }. Return 0/1.

board_cpu_reg(b, n): if (n < 16) return b->cpu.r[n]; if (n == 16) return apsr;
return 0.

board_enable_timetravel(b, stride, max_snaps): if b->tt is NULL, allocate a per-board
tt_t via tt_create_local(stride, max_snaps) (a new variant that does NOT touch the
g_tt global), store in b->tt. Subsequent board_run() builds a run_ctx_t with
ctx.tt = b->tt.

board_inject_irq(b, irq): public test hook -- builds run_ctx_t, calls
nvic_set_pending_ctx(&b->nvic, irq, b->cpu.cycles, &ctx). Used by smoke test to
provoke a TT event without needing a full firmware that hits an interrupt.

board_get_ev_log_count(b): return b->tt ? tt_ev_log_count(b->tt) : 0. Read-only
inspector for isolation tests.

board_destroy(b): jit_flush(b->jit); free(b->jit); free(b->uart_buf); if (b->tt)
tt_destroy(b->tt); free(b).

Implement src/lecerf_api.c (~80 lines): thin extern "C" forwards.

Constants in lecerf.h:

  #define LECERF_FAULT   2

Olympiad C: no asserts in hot path; calloc check returns NULL on OOM; b ? : NULL
guards on every public entry point.
